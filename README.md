# goodweather
一个简单的天气应用，使用了和风天气的数据接口，Material Design 风格

1. 可以罗列出全国所有的省市县。
2. 可以查看全国任意城市的天气信息。
3. 可以自由顺利地切换城市，查看其他城市的天气。
4. 可以手动更新及后台更新天气。

全国省市县数据 api 接口来自郭神提供的：
  http://guolin.tech/api/china

必应图片 api 接口也是来自郭神提供的：
  http://guolin.tech/api/bing_pic
